## Task 1 - Predictive Modeling (Classification)
Dataset: Telco Churn Dataset (bigml-80 train / bigml-20 test)
Tools: Python, Scikit-learn, Pandas, Matplotlib, Seaborn

### What was done:
- Preprocessed data (label encoding, feature scaling)
- Trained 3 models: Logistic Regression, Decision Tree, Random Forest
- Evaluated using Accuracy, Precision, Recall, F1-Score
- Plotted confusion matrix and feature importance
- Performed hyperparameter tuning using GridSearchCV
- Best Model: Random Forest
